Final Project Summary

Topic & Purpose

My website is dedicated to creating a Korean Learning Hub to help people learn Korean language and culture by providing resources.

Audience

The intended audience includes individuals interested in learning Korean, from beginners, language enthusiasts, and anyone looking to explore Korean culture through its language.

Design & Style

The design of the website will be clean, modern, and user-friendly, with a focus on readability. The color scheme will incorporate pastel colors, creating a visually appealing and calming style. Fonts will prioritize readability and achieve aesthetics.

Color Palette

Website Inspiration

  1. Visit Korea
  2. TTMIK (Talk to Me in Korean)
  3. Favorite Books, Films, & Albums

Scope

1. Home: Introduction to the website, quick access to resources.
2. Korean Language: This section will contain flashcards on how to count 1-10 in Korean (with examples, pronunciation).
3. Korean Culture: Explore Korean culture through music, cuisine, and travel tips.

Extras

JSON API Integration

Provide options for users to choose the Korean music or movie from the API. Allow users to set preferences, such as the theme or year of release. When users make a selection, use JavaScript to construct API requests and fetch data.

Event-driven DOM Manipulation

Allow users to create and interact with digital flashcards. They can flip cards to reveal the translation, pronunciation, or example sentence.
