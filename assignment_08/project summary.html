Final Project Summary

Topic & Purpose

The Reality Bar Webpage is designed to provide customers with a convenient way to search for and explore our cocktail menu. The website offers a platform for users to explore the bar's story, signature drinks, and engage in a visually appealing cocktail search.

Audience

The primary audience for this website includes potential customers to the bar. Designed for a diverse audience, the website caters to both those curious about the bar's background and seasoned cocktail enthusiasts looking for signature drinks.

Design & Style

The website adopts a modern and clean design with a user-friendly layout. Responsive design principles ensure a seamless experience across various devices. The color scheme, coupled with Bootstrap components, creates an inviting and visually pleasing atmosphere. The website aims to evoke the calm ambiance of Reality Bar while presenting information in an organized manner.

Website Inspiration

Scope

1. About Us: Introduction to the Reality Bar - includes the story, innovation, and contact information
2. Signature Drinks: This section will contain bar's signature drinks.
3. Search Cocktail: API database that return results of the user's input.

Extras

JSON API Integration

The project integrates with "TheCocktailDB" API, allowing real-time retrieval of cocktail data for the search feature.

Event-driven DOM Manipulation

The DOM manipulations dynamically render content, ensuring a smooth and interactive user interface. The website's navigation bar employs smooth scrolling, enhancing the user experience when navigating through different sections.
